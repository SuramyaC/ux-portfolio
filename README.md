# UX Research Portfolio

A clean, responsive portfolio website for showcasing UX research work.

## Structure

```
├── index.html        — Main page (all sections)
├── css/styles.css    — Responsive styling
├── js/main.js        — Navigation & scroll animations
├── assets/           — Add your images here
└── README.md
```

## Getting Started

1. Open `index.html` in your browser to preview
2. Replace placeholder text with your content:
   - Update your name, bio, and tagline in the Hero and About sections
   - Add your case studies with real project descriptions
   - Update contact links (email, LinkedIn, Twitter)
3. Add images to the `assets/` folder and reference them in the HTML

## Customization

- **Colors**: Edit CSS variables in `:root` at the top of `css/styles.css`
- **Fonts**: Swap the Google Fonts link in `index.html`
- **Sections**: Add or remove sections directly in the HTML

## Deployment

This is a static site — deploy anywhere:
- **GitHub Pages**: Push to a repo and enable Pages in settings
- **Netlify**: Drag and drop the folder
- **Vercel**: Connect your repo for auto-deploys

## Accessibility

- Semantic HTML structure
- ARIA labels for navigation and interactive elements
- Keyboard-navigable
- Respects `prefers-reduced-motion` for animations
- Sufficient color contrast
